How It Works

The Technology Behind PhishGuard

1. URL Submitted

You paste any URL into our scanner. It's immediately parsed and broken down into its components.

2. Feature Extraction

30+ features are extracted: domain length, TLD risk, suspicious keywords, encoding, subdomains, HTTPS status, and more.

3. RF Classification

200 decision trees vote simultaneously. The ensemble verdict — phishing or legitimate — is determined by majority.

4. Explained Result

You receive the verdict, confidence score, and a plain-English breakdown of every signal that influenced the decision.

How We Got Here

We built this platform to tackle the growing threat of phishing attacks using smart detection techniques. By combining machine learning with real-world URL patterns, we aim to make online browsing safer and more reliable.
